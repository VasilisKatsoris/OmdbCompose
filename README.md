
# OmdbSCompose

A showcase project that searches the OMDB database for movies

**Tech stack**
* Kotlin
* MVVM multi-module clean architecture
* Jetpack Navigation component
* Jetpack Compose
* Hilt-Dagger
* Retrofit
* Kotlin coroutines
* Flows
* Shared element transitions
* Paging
* Unit tests
